Demon Slayer Quotes API

Endpoints

GET/quotes

Returns an arrray of all quotes

https://demon-slayer-api.herokuapp.com/quotes

GET/quotes/random

Returns a single random quote

http://demon-slayer-api.herokuapp.com/quotes/random

GET/quotes/{num}

Returns an array of quotes (which are randomized) with length {num}.

For example, this will return 3 random quotes:

http://demon-slayer-api.herokuapp.com/quotes/3
